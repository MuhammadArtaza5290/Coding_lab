Computer:

At its core, a computer is a programmable electronic device that manipulates information (data).


Accepting information

This can be through various input devices like keyboards, microphones, scanners, or even sensors in smart devices.


Processing information

Using a central processing unit (CPU) and memory, the computer executes instructions (programs) on the data. This could involve calculations, comparisons, or manipulating text and images.


Producting output

The processed information is then presented in some way, like displaying it on a screen, playing sound, or sending it to another device.


Storing information

Computers can store data in various forms, like hard drives, solid-state drives, or even in the cloud, for later retrieval and use.


Components of computer

  • Monitor
  • Keyboard
  • Mouse
  • CPU
